README:
   This code provides a basic implementation of the Metropolis Monte Carlo algorithm
   for the Ising model. The code is written in Python and uses the NumPy library for 
   numerical computations. The main function metropolis_monte_carlo() takes a Hamiltonian 
   object and a configuration object as input and returns energy and magnetization 
   samples generated by the Monte Carlo algorithm. The BitString() class defines a binary 
   string object, which is used to represent the configuration of the Ising model. The 
   IsingHamiltonian() class defines the Hamiltonian object used in the Monte Carlo simulation.
